SA-MP 0.3e R2 server update - Kalcor - 13.07.2012

This update is for SA-MP server owners only

We are issuing a server update for SA-MP 0.3e to address some minor problems.

If updating from an existing 0.3e server: You can update simply by replacing the 'samp03svr' file on linux, or 'samp-server.exe' file on Windows, found in the 0.3e server download package. In this update, you also may need to update the samp-npc binary if your server uses NPCs.

SA-MP 0.3e R2 server

- Upgrades connection security.
- Adds server.cfg variable 'chatlogging' to disable chat being written to the server log. Add 'chatlogging 0' to your server.cfg to disable the logging of chat messages (These can be logged manually from your script).

Downloads:

SA-MP 0.3e R2 Linux Server: http://files.sa-mp.com/samp03esvr_R2.tar.gz
SA-MP 0.3e R2 Windows Server: http://files.sa-mp.com/samp03e_svr_R2_win32.zip
