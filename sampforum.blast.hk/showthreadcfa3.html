конечно есть, например такая:

сделать плагин который дал бы норм функции для работы с Map Icon's https://sampwiki.blast.hk/wiki/MapIcons

например я создал иконку

Code:
SetPlayerMapIcon(playerid, iconid++,   0.0,   0.0, 0.0, 1, цвет); // ID: ?? // ???
это тупо какойто белый квадратик какойто несуществующей текстуры (причём цвет этого квадратика не изменяется, хотя как сказано в вики он должен изменяться)

Code:
SetPlayerMapIcon(playerid, iconid++,   0.0,   0.0, 0.0, 0, цвет); // ID: ?? // ???
это иконка серый квадратик/треугольник типо как обозначение игроков/чекпоинтов на мини карте (этого id нет в вики там начинается с id 1) который видно на мини карте из любой точки (цвет также не изменяется постоянно серый)

так вот можно было бы сделать функции которые позволяли менять цвет для id 0, а также позволяли делать иконки видимые для игрока из любой точки карты как в сингле при миссиях + можно встроенный стример т.к. для некотоых серверов 100 иконок это маловато...

надеюсь ты понял о чём я
