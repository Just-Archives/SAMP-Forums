Good design, but I mentioned few things:
- Doctype tag must be the very first thing on the page (avoid even line breaks before it). Some (old, but still usable) browsers will not mention HTML5 doctype, if it is not on the first line.
- You included JQuery and Font Awsome CSS, but it's not used on the page - tell people, that they can use it (or put an example on test page)
- And if you wanted to put the main content in the center of the screen, you better try to use style "margin: 0 auto;". People have different screen resolutions, and on 1920 that page will be mostly on the left size of screen.

Overall, it's a good design idea - just put some more work in it, and it will be great.
