Main island module

- This module will only consist on one island, which will fully be completed by the time. It will consist on some partial detail elements but mainly the important things just to quickly get this module out first.

- More rural than the first island, and small compared to the main island. While they're both small, this island alone won't be as big as the main island but it will still be very compact and nice looking.

Details and bug fixes.
