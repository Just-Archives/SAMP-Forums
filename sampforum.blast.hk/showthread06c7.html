Dynamic In-Game Rule Maker [Mysql]
------------What is this?-------------

Well in the Filterscript/Include board i saw there is no Rule Maker. When i mean rule maker i mean as in as you see for some servers there are /rules where it will have the Server Rules. With this include you can create rules in-game by only using /createrule! This will help you to just make some server rules in game and not use "SendClientMessage" for your /rules command on your script. This is my first include so don't please go hard on me

------------Picture-------------


This picture just shows me using
/createrule
First then using
/rules
And after deleting that rule
/deleterule

------------How to Install This?-------------


------------------------------Installation-------------------------------------
1. Drag the Include folder to your script includes
2. Drag the plugins folder to your script plugins
3. Go on your mysql phpmyadmin then on top click Import and upload the .sql file
-----------------------------How to use---------------------------------------
You can get help by using /rhelp
COMMANDS: /createrule (rcon) /deleterule (rcon) /rules (everyone)



------------Questions / Answers-------------

* Question
Please Help, i keep getting "ERROR: You have reached the maximum ID" after i created 9 rules

* Answer
Well you just open Rules.inc and after you will see "#define MAX_RULES 10".
You just edit 10 and change the maximum you want
