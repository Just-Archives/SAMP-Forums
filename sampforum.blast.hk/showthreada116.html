Encryption is not possible. Why? Because the game needs the unencrypted data. You can't pass an encrypted model to the game because it won't know how to display it. You can send the model encrypted, you can store it on disk encrypted. But the moment you need to load it, you need to decrypt it. This means you need to be sent the decryption key. If you are sending the key to everyone, what was the point in encrypting it in the first place? They can either: 1) Use the key and decrypt it themselves or 2) wait for the game to load the model and pull it out of memory.

And don't say "yes, but only experts will be able to do that", because we all know that those experts will release a one-click tool for everyone to use within 7 minutes.
