Originally Posted by Mike861

It's a client update and has nothing to do with the server update - it will run with 0.3.7.

Kalcor has a different versioning system and it's the following way:

A new version is named 0.X.X where the X can be either an alphabet or a number (alphanumeric) and then he adds a RC if it's a release candidate and Rx where x is the release number.
And since it's the same version with just a different update for that release, it's R3 (or build 3) - so it'll work with 0.3.7.

That's it.
