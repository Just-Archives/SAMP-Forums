A little over a year ago, United Gaming decided to reopen with a fresh script. We began development shortly after the announcement. After a meeting with our Head of Development, we have made the decision to start a new script fresh as this script is really buggy and we won't be able to reach our deadline in time if we were to continue working with this script. We have decided to open-source the project to others whom have experience with roleplay scripts and can use this script freely, please do not use the United Gaming logo, name or alterations of the name to open your own community.

This script was transferred to sampctl, please download sampctl from here

Features:
0.3.DL

Everything is dynamic, this means that most features can be altered by admins IG.

Server Control - Perform MySQL queries, lock/unlock the server and change the servers hostname/password from IG.

Permission-based faction system - You can control which faction has access to what, whether it is /taser, /arrest, /dep, lockers, etc. This is a unique system that we haven't seen in other scripts.

Basic RP commands

Master Account System - Connect to the server with a normal nickname and have up to 3 characters, i think. (can be adjusted)

Fully functioning administration & helper system. Inlcuding a warn system, MySQL errors will appear to highest ranked admins IG.

Graffiti System - Spray certain spots, spots can be created by admins.

House System - Houses can be created by admins, not sure if there's a purpose for the house system yet.

Furniture System - You can decorate your house with furniture (The system is broken, not fully functioning but it's there)

Business System - Can be created by admins, has a stock system, business safes, etc. Cannot buy from a business with no stock. Not sure if there's a purpose to the safes except depositing cash, weapons & drugs & withdrawing them.

Custom Mappings - There are a few custom mapped interiors & buildings

MDC System - It isn't fully functioning but it's there. There's the ability to search players, via simple or advanced. Search a weapon(s) serial and check their licensing info.

Discord Connector - Can connect directly to discord

Boombox System - Self-explanatory, streams music through a speaker.

Vehicle System - it's pretty self-explanatory really.


I'm not sure of the other features really, but the script is pretty buggy and broken so you'll need experience in Pawn and sampctl to run it. If not there are sampctl tutorials somewhere.


We will not be providing script/bug support, however, I can assist with setting up if needed just drop a PM.
