LSPD Automatic Moveable Gates

Well After getting so much request on making moveable gates for the LSPD i decided to make it public and let people have the code.Its a very simple code and can be used to create other moveable gates all over San Andreas.All you need to do is change the co-ordinates,object id ect..This is my firts FilterScript so you may critize.I made this Filterscript mostly for New scripters as i know how hard it is to understand things from the beginning. You will need ZCMD in your includes folder for it to run.


Bugs Known:
Known at present but if you find any,let me know

download link:
http://pastebin.com/BbckUdE6

open a new file on pawno,delete everything and replace it with this code.Make sure to declare it on your CFG.

If you like it please +rep.
