Hey,
I have this gamemode lying around that i never can be bothered scripting more on, so I thought I'd release it. Maybe someone could find it useful.
I started on this a while back, when I wasn't as good as scripting. Also, most of the scripting was done when I was stoned.. There are a couple bugs here & there but allround I think it should run just fine.

To get you started..
Fire up the server, join, register, log in as RCON.
See /admin and /config.
To view bases, see /bases.
To start a base, /start base
To end a base /end
To add/remove players, /add, /readd, /remove
To spawn vehicles, /v
Ohhh and to test for pika, use /pika ... lol

All commands:
/resetskin -- /switchplayer -- /seat -- /pika -- /ping -- /viewbases -- /ot -- /beer -- /weapons -- /remove -- /readd -- /add -- /spec -- /unspec -- /f -- /uf -- /vehicle -- /auth -- /lock -- /unlock -- /me -- /setlevel -- /switch -- /sync -- /admin -- /config -- /config_line -- /config_reload -- /config_defaults -- /info -- /pm -- /kill -- /name -- /swap -- /startbase -- /stop

It's got a couple cool features:
  • Team HP graph after rounds.
  • In-game GUI config editor.
  • In-game GUI weapon menu editor.
See swg_h.pwn for more configuration options.
