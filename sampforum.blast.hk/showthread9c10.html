What does it do ?
You can convert the exact positions of multiple objects to the distance between every object and the player. You can save the converted positions in a file extension called .spawn

Now you can also load the .spawn file anytime/as much as you want, and enter a new player position.
Now it converts the distances back to the new positions, so you actually re-positioned your objects!

What is this usefull for?
(*)Clone your map without much work editing/re-making/adjusting rotations again, without touching a map editor
(*)Re-Position your map without touching a map editor
(*)Release as example your map with the .spawn file included so players can decide by themself where they want to create your map.

So basicly you only need to make as example an aircraft carrier in a map editor. And then make the spawn file like in the video (go to your map, and save your position nearby it and fill everything in)

AND NOW IF YOU WANT MORE/OR GIVE IT ANOTHER POSITION YOU CAN JUST SPAWN IT ANYTIME EVERYWHERE YOU WANT IT IN JUST A FEW CLICKS !

Once you got the spawn file, you can just enter the player position on the place that you want your map, and it will generate the pawno code for you!

It is actually a universal spawner creator (like the acc spawner above) creator... which can be created with this tool and also loaded to use in this tool.
