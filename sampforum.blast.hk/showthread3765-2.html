Cops and Robbers

This is not RP, You go rob casions and hold up stores you get wanted and the police has to catch you.

You have done a pretty good job.
I would like to suggest some improvements in this script if you are planning to update it:
Instead of a typing in a command like /arrest or /fine, you may use OnPlayerKeyStateChange to assign these to numbers which are easier to use, hence more fun.

I noticed that one player can do anything from robbing to raping. He doesn't choose a skill like robber or rapist. I'd suggest using skills that players have to choose so that they are everything but something. Also, adding few more skills like Hitman would mak it even better.

Keep it up.


EDIT: I tested this gamemode but there are a lot of bugs with checkpoints

For instance I went into the Four Dragons casino and the checkpoint is not showing up except on the minimap which says it's very high. -.- You haven't got co-ordinates right. Plus there should be points marked on the map where a player can rob, except the Caligula's and the Four Dragon's Casion there is no icon on the map.
