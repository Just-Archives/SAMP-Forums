UPDATE!
If you like Counter-Strike, you will love those gamemodes! I wanted to make them as similar to CS as possible.

For example, de_dam :
This is how it works:
The whole Gamemode is set and bounded to the Sherman Dam and it's sorrounding.
You have to choose between two teams, the Terrorists and the Counter-Terrorists. The Terrorists have to plant a bomb in order to blow up the dam. The Counter-Terrorist's objective is to defend the dam and defuse the bomb if it is planted. The round ends when the bomb blows up (Terrorists win), when the Counter-Terrorists defuse the bomb or when the Counters defend the dam for a certain roundtime, which is by default: 5 Minutes (both CTs win).

In my beta-tests many people wanted me to add Cars, but i dont want to do this, because as i said before i want to make it as similar to Counter-Strike as possible.

OK, this was the old explanation of de_dam most of you should know. Now the downloads of the very latest versions:
  • de_dam
    The classic de_dam map - but in a new version! Read changelog to see new features.

  • de_ship
    Same as de_dam, only at a different location. This gamemode is set on the aircraft-ship in Easter Basin, SF. Shorter ways for both teams, some very dark places and be careful with the explosives at the bombplace (cool feature though) !
    Downloads:
    [.amx]
    [.pwn]
    [DAntiCheat package]


  • fy_poolday
    This gamemode has not much to do with the two above. But it's still a bit like the very famous "fy_poolday" map in Counter-Strike. In the middle there's a big pool, and there are weapon-pickups everywhere. It's nothing than a huge war, just like real poolday from CS. In my first test with 5-10 people everyone told me, that he likes it - so imagine it with 20 people or even more! fy_poolday is perfect for a quick round of only killing to put away all your aggressions After tactical brain usage in my de-maps just chillout and kill some people for 10 minutes (that's the default roundtime).
    Downloads:

Changelog of de_dam:
Quote:

Changes from old de_dam version to this one:
° - stripped the code A LOT! (old code had 513 lines, new code has 458 lines but many more features!)
° - Added rounds-system! (default: 5 rounds б 5 minutes)
° - Added Team-Chat (usage: "![text]". [for those who don't understand: just a ! in front of your text])
° - Added Teamkill-punishment (Teamkillers diserve this!)
° - Added MANY sounds (such as the last five seconds of the detonation-countdown)
° - Improved detonation-countdown in general (old way just disturbed)
° - Fixed bugs like the gamemode crashing the server or timers not executing a function (thx to everyone who helped me! credits in the .pwn file )
° - Added "Intelligent-server-system": the gamemode counts the online players. If there are no players online for 30 minutes the gamemode ends. When players connect this timer is killed. If they all leave again the timer restarts, and all the stats like Team-Wins or bomb-plants are resettet.
° - changed MANY little things that are not necessary to mention here
