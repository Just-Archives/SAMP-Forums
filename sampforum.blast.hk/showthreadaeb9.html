Run Samp two times in One Computer ("very usefull if you are making a gamemode / fs etc")
Hello guys in this tutorial i gonna be showing you how to run SAMP/GTA in one computer , this is very usefull if you are making a gamemode / fs etc.. because you can test all by yourself in one pc , i making gamemode and i use this right now and it help me a lot and i want this to share to you guys

Ok lets start


1.Download the Sandboxie App
http://www.sandboxie.com/index.php?DownloadSandboxie

2.Install it
Follow the instruction or just click next until "Finish"

3.Go to Samp Folder
Right Click the samp.exe(Recommended! because in samp you can change your name in samp_debug not?) and you gonna see "Run Sandboxed"

4.After that there's a box will pop-up
Then Click OK ignore it just click ok

5.You can now debug in TWO PLAYERS! in one PC!
You gonna see on the windows title bar this
Quote:

"[#]SAMP[#]"
it means thats the SAMP you run using Sandboxie after that , open the Samp.exe without sandboxie and it will run without any error like Already Running Etc...

Done!...
