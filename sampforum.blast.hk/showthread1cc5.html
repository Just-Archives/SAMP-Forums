Well i had the same problem. All I did so it works again was this:
Go to your rockstar folder.
Right click on the samp.exe
properties > Compatibility > Change settings for all users
-Run This program in compatibility mode for: (Don't use this)
-Run this program as an administrator (Don't use this)
Now just start samp.exe by clicking right mouse Run as administrator. Or just run it without administrator.
Its works for me
