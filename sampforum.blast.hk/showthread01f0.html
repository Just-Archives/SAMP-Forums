Hi all,


After many requests, this project has been continued again. Because of that, Version 3 has been released.


What\'s new in Version 3:
  • + Unlike version 1 and 2, the whole site had a border so everything was in the middle. Version 3 now doesn\'t have those page borders anymore, the page goes right across with no gaps on the sides. This adds a much cleaner look.
  • + Added a donate section to the bottom of the page. You can add your own PayPal button underneath the box. (You can change the donation area to any other one you\'d like to.)
  • + Added brand new button styles. There\'s now a \'primary\' and secondary \'button\'. This can be seen in the demo in the intro area (right under the navigation bar.)


What\'s been updated:
  • + Brand new layout and design.
  • + Updated a much simpler navigation bar.
  • + Updated to a much simple looking footer.


What\'s been fixed:
  • + Fixed a bug where main texts wouldn\'t align in the center.
  • + Fixed the align bug with the footer. The footer now goes across the whole page.
  • + Fixed the navigation bug where the navbar wouldn\'t scroll while you scroll the page.
You can find the demo of Version 3 on the main post made before. Version 3 can be downloaded here.


I hope you guys like the new design. If there\'s any bugs please let me know and I\'ll fix them in Version 4\'s release.
