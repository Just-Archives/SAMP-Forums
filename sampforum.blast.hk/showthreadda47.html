This thread contains version 1 of the map editor.
Version 2 can be found here: https://sampforum.blast.hk/showthread.php?pid=3789899#pid3789899
Version 3 can be found here: https://sampforum.blast.hk/showthread.php?tid=648101

Features:
  • Toolbar - Stylish looking, mouse operated, easy to use.
  • Offset Editor - Set the offset of attached objects smooth & easy.
  • Model Browser - 20 list items / page, browse through everything or search for keywords.
  • Color Browser - 25 preview colors / page.
  • Vehicle Tuning - Fast teleports to & back from the corresponding modshop for every vehicle, unlimited tuning money.
  • Camera Mode - Travel around the map through solid objects.
  • Objects - Create, duplicate, remove, move, set textures, set colors, attach to other vehicles & objects.
  • Vehicles - Create, duplicate, remove, move, tune & set colors.
  • Pickups - Create, duplicate, remove & move.
  • Player Attachments - Create, remove, set model, set bone, set colors.
  • File Management - Save & load your maps easy & fast.
  • Ingame Information - Displays relevant information & keystrokes.
Screenshots: http://imgur.com/a/PHfF6#0

How to use:
  • 1. Download mapedit.pwn and the folder mapedit to your server filterscripts directory
  • 2. Download strlib & sscanf
  • 3. Compile the code
  • 4. Create the folder maps in your server scriptfiles directory
  • 5. Load the script on your server, and start mapping!
Download:
  • strlib (required) - https://sampforum.blast.hk/showthread.php?tid=362764
  • sscanf (required) - https://sampforum.blast.hk/showthread.php?tid=570927
