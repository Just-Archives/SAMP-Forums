Zombie Roleplay - SiM2hmoud - 07.05.2020

Introduction:
This is an early version of one of the Gamemodes i'm currently developing. This gamemode uses a unique idea of Roleplay mixed with Zombies which is pretty rare in SA:MP Scene. I've added the Admin system, a unused V.I.P system, saving system and inventory system and few more commands and features. The thing is i've stopped updating this gamemode cause it's too unoptimazed. Currently i'm working on a new version so i'm releasing this version (which is the first one) to public so other Developers can mess with it.


Features:
- Saving System with a Inventory system.
- A working Adminstration system with a lot of commands.
- A unused V.I.P system.
- Zombies system (NPCs).
- Basic Player Commands (/Rules, /help, /cmds...etc).
- Togglable Dialogs (Players can toggle dialogs so he can see them while using /cmds or he just sees text).
- A working Faction system.
- Materials & Chemicals system.
- Crafting system using Materials & Chemicals.
- Serverside Coins system.
- A improved Map of Los Santos... and a lot more.
