Halloween Zombie Mod v3
It's been three years since the last Halloween Gamemode I released, and it's been a really long time since I've released anything on these forums I'm planning on doing that again This is a really simple gamemode when I say simple it does not include the following

- Account System
- Admin System
- Ban System
- XP, Leveling etc system
- Anti cheat system (just money and weapons anticheat)

The reason behind that is, this is a really simple gamemode and it's only intentions are to be seen or probably an event for your server!

Now that's cleared lets move on to the features it includes,

- 50 Zombies this can be increased via the ZombieID variable (new ZombieID = 50
- Simple Money, Kill Counter, Death Message, Headshot, Death Animations, Weapon Shop system
- Random Spawns for humans and zombies
- Zombie Classes currently only two and the rest are defaults, the two special zombies are Boomer Zombie which blows up upon death (clown) and the homeless zombie that runs fast
- Earn money by killing zombies

There are quite more features you can view them in the video.



https://www.youtube.com/watch?v=rTUt4sQsWBY

Plugins Required:

Sscanf, RNPC

Includes Required:

foreach, rnpc, zcmd, y_timers, y_ini, y_flooding

Source:

http://www.pgagaming.net/forums/inde...sa=view;down=5

Enjoy lads (:

(not my best scripted gamemode)
