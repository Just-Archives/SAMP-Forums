As the title says, I'm releasing my Hydra Dogfight script.
It has 2 teams(USA(AREA 69) vs Russia(SF Naval Base)).
First team which gets to 50 kills, wins, but if none of the teams could get to 50 kills in the 30 minutes, the team which has the most kills wins.
BUT, if the kills are equal/the same, then the team which owns the Los Santos Airport wins.

Each player spawns on a Hydra which is located in the teams homebase.
There is a dogfighting script, so you actually get kills etc. from the Hydra kills.
I've included a some-what basic Admin System & Register system(SQLite).
There are enough commands to control the server, atm. there are 29 admin commands, and 15 player commands.

So, reading from ^, I hope you've realized that this script is a pretty 'straight forward' one, which doesn't have "flashy" features - It's just made for dogfighting.
