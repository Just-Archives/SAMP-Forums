Quote:
You can disable a fix of any of the places at the beginning of the filterscript or before the include.
For example, write following: #define DISABLE_MAPFIX_PLACE_1 // So it will disable the fix 1
How to install:
  • Filterscript version
    1. Download mapfix.pwn from the links below to the filterscripts folder
    2. Open the downloaded script and compile it
    3. Open server.cfg and write mapfix in the filterscripts line
  • Include version
    1. Download mapfix.inc from the links below to the pawno/include folder
    2. Open your gamemode and write the following: "#include <mapfix>"
