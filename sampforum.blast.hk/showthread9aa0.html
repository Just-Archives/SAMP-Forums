No, that isn't possible.

But I believe this is possible:
pawn :
new color[] =
{
 0xFFFFFF,
 0xFFFF00,
 0xFF00FF,
 0x00FFFF,
 0x0000FF,
 0x00FF00
};

// @ /color [number]
SetPlayerColor(playerid, color[number]);
