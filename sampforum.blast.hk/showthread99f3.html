Jueixs House System V2

Hello, I have recently decided to leave sa-mp but still release scripts every now and then. I have decided to release the house system for more server (Note: This is not the full verison it will get updated to the full verison slowly to work more as a filterscript.) seen as I don't want any of my systems to go to waste and maybe someone else will have a use for them.

Features:
  1. House creating in game.
  2. Browse through all GTA San andreas Interiors in game and add them to the house, Browse interiors with no furniture and allow players to create houses from the bottom.
  3. Custom Plane for players to 100 percent build the interior, Walls, Doors And more.
  4. Custom made furniture selection menu, Allows for easier furniture selecting and searching.
  5. User friendly, Easy to add more furniture to the lists.
  6. More features coming soon.
Commands:
  1. /createhouse - Opens the menu to allow you to create the house.(Rcon Only)
  2. /houseints - Allows you to view all interiors in game and set them as the house interior (Rcon Only)
  3. /furniture - Allows the player to put furniture into the house.
  4. Go up to a house and press H to buy and enter it.
