VERSION 1.1 (PROMO)
(Release Date: March 18, 2007)
Information: Yes, it is finally here; the spoof of the next version with a big improvement from the first version. If you though that was a lot, you'll be astounded by the added features of this one. Here is the new interface with a preloaded example:

/imageshack/img239/9...11promoyn8.png

Features: Here are the features for this version:
  • A higher quality image in the GUI.
  • Status label fixes when saving.
  • Save file bugs have been revised.
  • Form resized to include a greater variety of features.
  • Blip size can now be as low as 2 and as big as 30.
  • Mini description of the best feature to yet be upgraded.
  • A box which displays the car's id, car type name, car type color, and car type occurances.
    • The ID number only occurs once per car type and displays the car type ID that was used in the textbox where you entered your addstaticvehicle statements.
    • The name (not all names added yet, which explains the blanks), displays the full car's name corresponding the the car id.
    • Blip color can be explained as each car type gets its own color on the map. To change it, click the color (all of them defaulty light blue), select the color you want, then update the image again.
    • The occurances is the amount of times that car type has been added in the script.
    • NOTE- YOU CANNOT CHANGE ANY OF THE VALUES IN THIS BOX EXCEPT BLIP COLOR.
    • Finally, the output box displays the number of car types used, cars used, and the max values you can put for each.
    Also, the compressors for the images have been improved and the image quality itself has been improved. Below is the saved image displayed in the spoof image:

    The release will have a little more features and will be released tonight.

    Beware: The source is going to be up to 60 MB and the app 7MB.


    Enjoy! Hope to see some comments .
