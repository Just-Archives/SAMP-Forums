CRIME STATE: LOS SANTOS COPS AND ROBBERS
Hello,this is my gamemode that i have been working on with X337 about one year ago,I left this project on August 2015 if i'm not mistaken,and one year later I found this script on my hard disk and decided to working on it again and put it online(you guys can see it on server advertisement section).it's not really success,so yeah good luck with it,sure that I got permissions for X337 to release it.

This script is pretty mess up,i'm too lazy to optimize them.
there's filterscript that I found awesome on this forum and I added it to gamemode.

THIS IS NOT A CNRSF NOR CZCNR EDIT

Note: *red means bugged

Features
  • House system
  • Business system
  • Garage system
  • ATM system(took it from rootcause's script and i was edited it,you can shoot them)
  • Lotto system
  • *Gang system
  • Token system
  • Coin system(similar to token system,but you will automatically get it after playing 1 hour)
  • Hot Property system(rootcause's script)
  • Gun Vending Machine system(I edit it,rootcause's script)
  • Colormatch Minigame(rootcause's script)
  • Gate system(rootcause's script)
  • Drugs system(rootcause's script)
  • Fuel and Fuel station system
  • Jobs System(Drug Dealer,Kidnapper,Terrorist,Rapist,Hitman,Gun dealer,Mechanic)
  • Rooftop system
  • Singleplayer bribe system
  • Fishing system
  • Trucking mission,Forklift mission,sweeper mission,pizza mission
  • VIP system
  • GPS system
  • Advanced robbery,you have to aim the actor
  • there is more and more systems you can find
