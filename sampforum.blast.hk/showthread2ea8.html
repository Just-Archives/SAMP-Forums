« ZOMBIE APOCALYPSE » (SA-MP 0.3e)
Best (T)DM Script
My First Release So Please Have Some Kindness
REP+, SUGGEST, COMMENT




«INTRODUCTION»
Hello guys! I am pretty nice in scripting as i have script this zombie apocalypse server once released and was hosted in samp 0.3c by pond a good scripter. So i thought to make it and host but i further thought of a Gangwar server so i released this Gamemode on Samp Forums for all people kindness XD. This is pretty advanced script which is having all the best functions preset in it. All the best zombie gamemode by me. It contains real movies effects with infections and zombie and human classes within powers. But this is incomplete yet. So you have to edit this script for proper gameplay. You have to just make few class functions written in the /class dialog. So enjoy and read the FEATURES section above for MAX_Info.




«FEATURES»
- A Fully Configurable script. ( I mean little bit )
pawn Code:
#define Mode                    "BETA (v1)"// Mode name
#define Website                 "www.sa-mpdm.co.cc"// Your forums / website URL*
#define Servername              "« ZOMBIE APOCALYPSE » (SA-MP 0.3e)"// Your server hostname
#define Owner                   "Q_Lite"// Server owner name
#define ZOMBIE_CLASS_SKIN       181// Official zombie skin on Request Class
#define GAME_TIME               360000// 1000 = 1 Second, 60000 = 1 Minute
#define GAME_SECONDS            5
#define GAME_MINUTES            300
#define HUMAN                   1
#define ZOMBIE                  2
- 2 Teams, with Zombies & Humans.
- We can only select Zombies on Team selection on the Call back of Request Class.
- Infection system with a blood fader or screen fader with red effect.
- Zombie classes, Total = 9.
- Human classes, Total = 10.
- Switch class Selection by /Class command with ZCMD.
- Uses of all the latest includes.
- 6 Scary Maps fully edited by me for max fun.
- Anti - Team kill system.
- Random infection system, although its not kitten's random infection system.
- Textdraw for each team to tell their objectives through messages.
- Player XP system with Textdraw support with white color in right upper corner.
- Class system bassed on player XP count.
- TOP5 player on map ending with textdraw and player get to spectating mode ( For 10 Secs ).
- Mission Draw, Complete & Failed system ( + 10 XP to winner )
- Death messages with 10 XP + gametext
- Boomer Zombie class explodes on death.
- Instant class switcher on every request Spawn.
- Infections with Standard, Blind, & Drunk Visionary ones.
- Key state infection causer. When you press 'KEY_WALK' the player infects the nearer human and performs a Anim to show its working.
- Engineers can build Boxes by pressing 'KEY_LOOK_BEHIND'.
- Rebot can Build TNT By pressing 'KEY_LOOK_BEHIND'.
- The reaper Zombie haves special reper attack.
- Class Mi-7 Can Reload current weapon by pressing 'KEY_LOOK_BEHIND'.
- A new 5 minus timer with textdraw.
- Team statics textdraws.
- Class switching for NEXT SPAWN which is very cool feature. It saves player time for switching classes on Spawn.




«COMMANDS»
/Class - Switch Class for next map ( If Instant then spawn player ), Classes available for Zombies and Humans.
/Human - Test command to switch to human Team.
/Infect - Test command to switch and test infection.
/Drunk - Test command to Switch Drunk vision Infection.
/Blind - Test command to Switch Blind vision Infection.
/Map - Test command to Switch Next Map.
/Stop - Test command to Switch Infection OFF.
