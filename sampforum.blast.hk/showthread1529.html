How To Install?
You need few includes:
  • a_samp
  • Dini
After adding these includes to your "Pawno\includes" folder, Simply add this FilterScript to your folder and server settings
  • Server CP.pwn Or Pastebin Link
  • Server CP.amx
And then this is the executable file:
  • SAMP Server CP.exe
And the final step is to create a folder with "CPFolder" as a name in your "scriptfiles" folder. Where the objects will be stored.
