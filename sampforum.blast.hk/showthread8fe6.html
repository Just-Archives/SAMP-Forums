What is this?
It's a MySQL Admin System.
Actually, it's RealCop228's "Clean Script". I decided to edit it, and add some new features and bug fixes!
Original Topic: Here

The new features:
  • It's no longer a gamemode, you can use it as a filterscript now!
  • Added Auto-IP Login
  • Fixed saving IP bug( it was saving every IP in the database as 255.255.255.255)
  • Added user counting and Account ID displaying(if someone registers, it will show the total of the users found in the database.)
  • Added /stats command to show the stats of a specified player.
  • Removed secret question/answer stuff(it wasn't really needed)
  • More which I can't remember right now
