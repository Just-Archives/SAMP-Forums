GTA Gaming Gangwars

Information

I opened a server with few friends but later on two guys start doing stupids things like hacking car spawning just for a small reasons and now I m tired of this server. If you guys found any major bug please report I will try to fix it. Script is completely created from scratch but some codes are taken from open source code available on forum..

Note: Its an old school script.


Features

8 Teams for normal players.

Each 8 teams have a gangboss

Separate Team for Administrators

Separate Team for Donators

In-Built AAD System
Server has an In-Built attack and defense system. Administrators can organize the Attack and Defense TDM mini events.

Experience System

Duel System
In built feature to duel from your rival. You can chose weapons, locations and stake your money on the duel.

IRC System
Not 100% completed.

/shop system
Allows the players to purchase weapons and ammunition necessary for combat. You can purchase armour and health by using simple commands.

Ban system
There are 2 type of ban system:
1. IP Ban System
2. Router Ban System

You can offline ban anyone from ingame using /oban command.

Anti Cheat system
There are anti cheat system for weapon cheat, Jetpack and Fake kill.

Both are efficient as tested.

House System

Property System

Private Vehicle System

Donator System

Turf Capturing System

Admins are able to manage shop price from ingame.

Money Bag System
Start only when admin triggers it.

Anti Death Evading System

Download and Installation

Instructions

Download server package
Create a MySQL database
Upload the .sql file (dump.sql) to your MySQL server
Type in the correct details within the .pwn to your MySQL server
Compile script
Run server
