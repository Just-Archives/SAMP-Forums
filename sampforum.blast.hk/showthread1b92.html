Greetings to all users of the site, this time I present to you different genres modes High Speed ​​Drivers. This mod has five genres: Drift, Stunt, DM, TDM, Parkour.


What is in fashion:


- The system of registration / login
- System Administration:
* Trial Admin (Level 1)
* Basic Moderator (Level 2)
* Moderator (Level 3)
* Master Moderator (Level 4)
* Administrator (Level 5)
* Master Administrator (Level 6)
* Professional Administrator (Level 7)
* Server Mapper (Level
* Server Beta-Tester (Level 9)
* Server Owner (Level 10)
- System VIPs:
* Bronze VIP (Level 1)
* Silver VIP (Level 2)
* Gold VIP (Level 3)
* Premium VIP (Level 4)
- Many different entertainment zones
- There are seven "worlds":
* World Drift (Drift World) - Designed for drifting, in "this world" a lot of places to drift
* World constant (Stunt World) - Designed for extreme, in this "world" a lot of places for constant
* World Skill (Skill World) - Designed to test your skills, if you finish a call parkour, get some cash and glasses
* World DM (DeathMatch World) - Specially designed for the single "massacre", kill as many enemies as you can in special places DM
* World TDM (Team DeathMatch World) - This world is created specifically for the command "massacre", but together with his team, you can destroy all your enemies
* World admins (Admin World) - Specially designed for managers, where they can discuss with each other while being in a quiet room
* World VIPs (VIP World) - Specially designed for VIP-persons, where there is room and Vip Vip Tower, the tower many floors


Commands:

Код:
Commands for administrators:
* Level 1:
- / Disarm [playerid] - removes all the arms of the selected player
- / Clearchat - clears the chat for all
- / Stophns - unknown
- / Unjail - pulls a player from jail
- / Jail - puts the player in jail
- / Adminduty - unknown
- / Aweaps - allows the administrator weapons
- / Spawnplayer / spawn - spawn car / player
- / Tp [x] [y] [z] - Teleport to the chosen coordinates
- / Healall - treat all
- / Unmute - unknown
- / Ann / announce - sends a message to all GameTextForPlayer
- / Goto - Teleport to the selected player
- / Unfreeze - unfreeze player
- / Setskin - change skins
- / Get - teleport a player to you
- / Explodes - explodes vybronnogo Player
- / Acar - creates machine
- / Abike - creates a motorcycle
- / Aplane - creates aircraft
- / Aheli - creates a helicopter
- / Aboat - ssozdaet boat
- / Warn - warn the player, 3 warnings = kick
- / Showrules - shows the rules for a player
- / Acmds - shows the list of administrative commands
* Level 2:
- / Disarmall - remove weapons from all players
- / Slap - slap a player in the air
- / Spectate - Follow player
- / Stopspectate - stop watch player
- / Setworld - sets the specified virtual world for players
- / Setinterior - sets the specified interior for a player
- / Kick - kick a player
- / Port - the same as / goto
- / Teleplayer - teleport a player to another player
- / Akill - kills selected player
- / Settime - sets a specific time for a player
- / Setweather - gives the player a weather
- / Admins - shows a list of online admins
* Level 3:
- / Starthns - unknown
- / God - enables / disables God
- / Jetpack / jp - gives you the selected player or jet pack
- / Sethealth - gives health to a certain player.
- / Mute - unknown
- / Respawnallvehicles - spawn all vehicles
- / Freeze - freezes player
* Level 4:
- / Giveweapon - gives weapon selected player
- / Ejectall - throws all players from cars
- / Eject - ejects a player from vehicle
- / Setscore - sets the Player Points
- / Setmoney - sets the player money
* Level 5:
- / Kickall - kikaket all players
- / Getall - teleports all players
* Level 6 - 10:
- / Ban - ban a player
- / Rban - unknown


Commands for VIPs:
- / Cc - change the color of the car
- / Boot - opens the trunk of a car
- / Bonnet - opens the hood of the car.
- / Lockcar - closes the car door
- / Unlockcar - opens a car door
- / Lights - unknown
- / Speedboost [on / off] - Unknown
- / Plate [text] - change the number of car
- / Engine - enables / disables the engine
- / Viplounge - teleport to VIP - room.
- / Viptower - teleport to Vip - Tower
- / Vcar - unknown
- / Vbike - unknown
- / Vplane - unknown
- / Vboat - unknown
- / Vheli - unknown


Teams for players (Sami translation of the description!)
- / Santahat / shat / sh - puts a santahat on your head.
- / Joinhns - joins the Hide n Seek event if there is one arranged by the admin.
- / Car [carid / name] [color1] [color2] - spawns the selected car for you.
- / Fstyles - shows a list of fightstyles, for the DM world only.
- / Repair - repairs the car, pressing the H key will also do this and flip your car, there is autorepair in the server by the way.
- / Flip - flips your car.
- / Sorry / forgive / poop / hate / noob / rape - Troll Commands.
- / S - saves the current position your at.
- / L - loads the position you saved last.
- / Carcolour - changes the carcolour of your vehicle.
- / Savethisskin / saveskin / useskin / skin - saves the skin you have, saves the skinid you want, uses the last saved skin, sets your skin to the selected one.
- / Changeworld - change your world with this command, example: your in Drift World and want to go to Stunt World, do / changeworld and select Stunt World.
- / Radio - some radio stations here.
- / Givecash - gives cash to the selected player.
- / Heal - heal yourself, only works outside DM / TDM World.
- / Countdown - makes a countdown for everyone set from 5.
- / Races - shows a list of racenames available.
- / Report - reports a player.
- / Suggest - sends a suggestion to all online admins and it gets saved into a file in scriptfiles, so the owner can see what improvements can be made.
- / Para - gives you a parachute.
- / Me - sends a message to all.
- / Say - same as / me but now with Says: in it.
- / Color - list of available PLAYER colors.
- / Night - makes it night.
- / Day - makes it day.
- / Kill - suicides.
- / Rules - shows a list of the rules.
- / Savestats - saves your stats.
- / Stats - shows your ingame stats.
