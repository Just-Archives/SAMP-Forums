SA-MP 0.3.DL

SA-MP 0.3.DL is a seperate version of SA-MP featuring server-side custom models. Right now, custom objects and custom player skins are supported, with more types planned for the future.

This version was branched from SA-MP 0.3.8 development.

Updates:

SA-MP 0.3.DL R1 Client/Server

- Imports everything from last 0.3.8 RC (RC4-4)

Downloads:

0.3.DL-R1 Client
SA-MP 0.3.DL-R1 Client Installer

0.3.DL-R1 Server

Please review the SA-MP Services Agreement if you wish to run a public SA-MP server.

SA-MP 0.3.DL-R1 Windows Server
SA-MP 0.3.DL-R1 Linux Server
