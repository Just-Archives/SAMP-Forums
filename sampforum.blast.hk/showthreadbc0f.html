How to make yourself an admin:
  1. Create an account.
  2. Exit the game.
  3. Go to the game files, Scriptfiles > Accounts > Firstname_Lastname.ini.
  4. Edit the line says "AdminLevel" and set it to 5. (5 is server owner don't go above it.).
  5. Relog back.


How to change your admin name that presented in /admins?
Open the 'PRP.pwn' using a pawno.
Click CTRL+F (Find).
Find "stock GetAname(playerid)"
You will see things like:
if(PlayerStat[playerid][Aname] == 1) AdminName = "Marco";
Edit the 'Marco' to anything you want.
To add more slots just add another line with a new ID (the ID is colored green above).
Make sure there's no two identical ID's.
