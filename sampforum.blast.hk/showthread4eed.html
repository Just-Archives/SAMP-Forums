Me, I started coding casually. After a playtime of 3 hours on a TDM server, I went exploring my SA-MP folder. And when I found pawno.exe I was like
"What the heck is this.. A .exe file in SAMP folder.. Should be some sort minigame developers leave in game folders"
I still remember the feeling when I opened that horrible editor. Never ever thought it'd catch my attention that much. That "build" button ended the job in the end. It made me realize it was something to create gamemodes!
The only thing I was able to do was to SendClientMessage with odd colors and use SetPlayerPos.
I play SAMP since some years, 2013. If only I would have opened that .exe earlier.. Maybe I'd have taken another path in my life.
