Shifty's Freeroam - Let's Get Started
Introduction

I've made a simple Let's get Started script fully by myself, at the beginning it was about going to be my own server but I canceled the project, there's not much special about this script but it's handy for players that are wanting to start something that are lazy to write all the basic things or for players that are new with scripting.

General Information

I've fully created this gamemode from scratch, except the group system, [HiC]TheKiller has made the group system which is implemented in this gamemode, the gamemode is a freeroam based gamemode with a few commands. So far as I know there are now bugs into this script. I've tried to mark everything with an explanation but if there are any questions feel free to ask them in this thread... If there is enough zest for this then I will keep developing and add new things..

Credits

Scripting: Shifty
Group System: [HiC]TheKiller




Features and options

Build 1
  • Some command with Y_CMD
  • Custom made Car Defines
  • Group System ([HiC]TheKiller)
  • Random Messages, every 10 minutes
  • Auto-Repair
  • Speedboost, Jump and Flip

How to install

1) Download the RAR package.
2) Make a folder at your desktop called 'SA-MP server' or give it your own name.
3) Copy the rar file to your desktop.
4) Double click on samp.exe and the server will open.
5) Go to http://www.whatismyip.com/ and put your IP at the SA-MP launcher (default port)
