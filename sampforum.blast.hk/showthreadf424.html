RPUCP
(roleplay user control panel)
Features
  • Dashboard
  • Characters selection
  • Each character's page that displays a lot of info
  • Login/Logout system (With Sessions)
  • MySQL support
  • PDO optimized queries and connection
  • Whirlpool encryption on login page
  • Skins display

License
  • You are free to edit it and use it for your server if you credit me
  • You can't re-upload and claim that the UCP is yours
  • You can't sell this project to someone
In case I see anyone to abuse the license, the download link and thread will be removed

How to use/Install
Well, this UCP is not suggested to be used from newbies on PHP coding, it requires some knowledge.
So if you are a newbie just skip it. Now if you still wanna use it you will need to edit the MySQL's
connection to your server's database, both of databases must be in the same host or you will need
remote MySQL connection. So after editing MySQL's connection stuff you will need to edit the queries
in every single file (Tables, fields etc.), this is why I am saying that it is a bit hard to use.
