Server can restart if there are many players, even when with just a couple of players it was fine.
You should run apart from the server log another log that logs all the players text and commands. If you find that the server restarts after a certain command or text, check your script on that one.
Otherwise a server can restart because of buffer overload, especially with heavy scripts.
