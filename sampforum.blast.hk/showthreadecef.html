If the server owned a private/public key pair, it could encrypt the model with the private key. It will then rest in the cache folder, and SA-MP will take care of decrypting it and then loading it into memory. Of course, it would not keep experts away, but all in all, custom maps are also potentially able to be stolen, and yet I haven't seen any instance of this ever (nor a "hack" to allow it).

As for the rule, with all due respect, I am not going to feel obliged to abide by it. Common sense and morality is my rule, not semi-legal restrictions.

After all, there has been CUSTOM.IMG for eternity, and nobody cared about the laws. There is no difference if the models are to be downloaded from an external site or from the server directly, it's just a matter of convenience.
