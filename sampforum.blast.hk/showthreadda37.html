TRUCKMANIA
International Trucking Server
176.31.192.104:7777


If you like trucks - this server is for you! Truckmania - the first, unique international trucking server that redefined the meaning of trucking in SA:MP, setting standards that almost every other trucking server actually follows welcomes you, with script full of features, tons of fun, big community and no complicated or unnecessary features, to experience fantastic world of 18-wheelers (and more!)

SERVER FEATURES:
- unique script built from scratch with as little bugs as possible
- many mapped locations (13000 objects), including truckstops, truckyards and 13 new islands
- 6 different classes (Trucker, Courier, Bus Driver, Assistance, Police and Mafia)
- delivery missions for Truckers, Couriers and Mafia
- truck and bus licenses
- about 500 selectable missions for licensed Truckers available
- brand new convoy system allowing to drive together with your mates
- Cement Truck, Securicar, Firetruck and Pizzaboy minimissions
- 21 bus lines in Los Santos metropolitan area - full-featured bus system
- towing, repairing, refueling cars for Assistance
- balanced Cops'n'Robbers style Police system not giving Police any unnecessary authority, allowing anyone with sufficient score to chase, ticket and arrest suspects
- trailer hijacking missions for Mafia (F'n'F style - detach it, attach to your truck and return to base)
- ranks system featuring 5 ranks for every class available on server
- companies system allowing creating or joining professional transport firm
- accurate speedometer
- fuel system (refueling needed)
- CB radio with 40 channels and assortment of 10-codes
- hidden trailers - collect 20/20 to earn 500000$ and special privilleges
- speedtraps
- car renting
- houses with house cars
- events including weekly convoys hosted by admins and The Event Team
- many more details, like overloading, infobar, trucks don't explode etc.

WHY US?
- experienced, big admin team that doesn't allow bad behaviour on server
- we were the first server (introduced in June 2009) that gone international and broke Polish servers' scheme of "drive around a map without purpose and knowledge what you haul" by making unique gamemode that have been carefully developed for months, soon followed by other servers
- you aren't forced to do missions selected by script - you can select any load and route available (more for licensed truckers)
- our server is still under development, there are many unique features coming in next few months
- you aren't forced to keep to the roads - if you want, you can go offroad trying to find faster way to deliver cargo (driving wrong way is still prohibited)
- quality mappings done by talented mappers, including famous Eric's Truckstop and massive islands chains
- weekly events prepared by our Event Team

OTHER INFO:
- not-flicking truck mods are allowed
- absolutely no killing/beating/ramming/shooting on server!
- this is not RP server, but roleplaying is welcome
- server is protected by custom anticheat and experienced admin staff
- server is constantly updated, soon to come: load redesign and new, unique hauling mode
- server forum:www.truckmania.org
- teamspeak 3 30-slot server available at 78.129.193.205:10040
