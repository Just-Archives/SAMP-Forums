Just a simple CTF i was making for a new server, but then i decided to switch to RP With luciano.
it has a mini admin system using dudb and sscanf, with several commands, more can be easily incorporated.

Basically, theres 2 teams.

Alpha and Bravo. You try to get each others flags.
Each player spawns with 1000 rounds of UZI in their base.

Im basically releasing this for anyone who would want a base of something to work off of , or someone who would like to see some DM examples and stuff.

Heres a feature / cmd list.

Player cmd's
/stats
/login [pass]
/register [pass]
/help

Admin Cmd's
/setlevel [playerid] [level]
* iLinx slaps [id] around a bit with a large trout.
/kick [id]
/ban [id]
/veh [carid]
/atele
/btele

Features
Drop Flag
Pickup Dropped Flag
Flag object attached to player

Credits
Dracoblue -> The register/login tutorial, i just kind of copy and pasted that dudb code :P , also for creating dudb
Credits to the makers of DCMD , SSCANF and CPS ( the checkpoint streamer used )
Sorry if i missed anyone else

How to make yourself admin
Register on server,
Go into scriptfiles->users
open your file
change adminline to desired level [1-10]

Download Package
Pastebin

Keep the credits ?

Mirrors are accepted

Post bugs here and i'll see if i can fix them for you x)
