If you always crash with this palm tree object:
Code:
SCM Op: 0x107, lDbg: 1 LastRendObj: 620
Then there might be something wrong with the object model (ID: 620) and 0x107 indicates object create function...

If other players don't crash like you, try to reinstall your game (clean) and sa-mp.

Otherwise the server is creating too many objects.
