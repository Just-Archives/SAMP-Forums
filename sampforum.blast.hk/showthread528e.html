MrTrane's SA-MP Teleport Maker
------------------------------------------------------------
Description:

MrTrane's SA-MP Teleport Maker was made in Visual Basic Programming language, the SA-MP logo belongs to the SA-MP Team and the San Andeas footer to it's creator (kc). I'm planing to make a new version in the future which you will be able to export the result code into a filterscript file and with more advanced features and benefits.

Pictures:


Features:
  • SA-MP Logo: Link to the official SA-MP website (www.sa-mp.com).
  • Name: You may enter the command that will be used to teleport to the location.
    Coordinates
  • X: PosX coordinates.
  • Y: PosY coordinates.
  • Z: PosZ coordinates.
  • Facing Angle: Facing Angle.
  • Interior: Interior ID - (0 as default).
  • Virtual World: Virtual World ID - (0 as default).
  • Message: SendClientMessage - (leave it blank if don't want to send any).
  • Message Color: SendClientMessage Color
  • "Or download Color Picker Tool(link)": Link to RyDer's SA-MP Color Picker Tool (https://sampwiki.blast.hk/wiki/Color_Picker_Tool).
  • Player: If Player checkbox is selected it will teleport only the player.
  • Player with Vehicle: If Player with Vehicle checkbox is selected it will teleport the player with his vehicle.
  • Generate: Generates the code into the big TextBox.
  • Clear: Clears the code TextBox field.


    Available Colors

    Red - 0xFF0000FF
    Pink - 0xFF80FFFF
    Blue - 0x0000FFFF
    Green - 0x3DD926FF
    White - 0xFFFFFFFF
    Orange - 0xFF8000FF
    Purple - 0xCA1CDDFF
    Sky Blue - 0x4EDED7FF
    Yellow - 0xFFFF00FF
    Grey - 0xAFAFAFFF
    Light Red - 0xF95572FF
    Light Blue - 0x1CA2EAFF
    Light Green - 0x80FF80FF
    Dark Red - 0xD50000FF
    Dark Blue - 0x1709A2FF
    Dark Green - 0x1BB412FF

    - There is not a limit to the colors that you can create and use in your messages, download the SA-MP Color Picker Tool and makes your owns, I just added the most popural and basic ones.

    Download:

-Free on comments and suggestions, this is my first release.
