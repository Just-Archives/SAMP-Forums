RSign - SA-MP Server Signatures
ABOUT
RSign is a free SA-MP Server Signature generator. You can easily customize (language, image, colors, fonts, ...) the signature you want to generate.
I know that the design sucks but it's made to be customizable, so change everything like you want !

Author
This project was developped by Rafael 'R@f' Keramidas <rafael [AT] keramid [DOT] as>.

Donations
If you like this project or make a commercial use for it, please donate via paypal at rafael.keramidas [AT] gmail [DOT] com. Of course you don't have to but it would be nice if you do

Credits/Thanks
WhiteRabbit Font : Made by Matthew Welch. Check out his awesome fonts at http://www.squaregear.net/fonts/

PREVIEW
You can try RSign at : http://rafael.keramid.as/rsign/
or


DOWNLOAD
You can get the project on Github : https://github.com/RafaelKeramidas/RSign

LICENCE
This project is released under GPLv3. Read the GPLv3.txt for more information.

REQUIRED
LibGD & Sockets for PHP.

INSTALL
Extract all the files in your folder and change the config file (includes/config.inc.php).

SUPPORT
I don't give ANY support for the installation or usage of this script, so please don't contact me for that !!
If you found a bug, write me at rafael [AT] keramid [DOT] as and don't forget to put "RSign" in the subject.
