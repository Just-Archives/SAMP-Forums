Xenon Gaming Roleplay

Hello! This is my first gamemode / major release. This gamemode is nothing extraordinary, how-ever I plan on making it more than the average roleplay gamemode. The gamemode as of now is only a skeleton of what it will be and is in no way complete. This game-mode is based in Fort Carson. It should be noted that if an update modifies the SQL file you'll be able to update the rows / columns by downloading the "update.sql" file(not yet applicable).

Scripting Notes:
[*] Never use SetPlayerSkin when setting a players skin. Always use SetPlayerSkinEx, skins are saved by ID not by the players current skin.

You will need the following includes:
* Default SA-MP Includes *
* MySQL Plug-In/Include(I use r33) *
* Streamer Plug-In/Include *
* Foreach Include *
* ZCMD Include *
* SSCANF(2) Include/Plug-In *


You can find the latest version here:
https://github.com/Abagail/XenonGaming

Recent Changes:
- Unified the error messages into one function
- Unifed the syntax messages into one function
- Added a simplistic(untimed for now) ban system with the commands /ban and /unban.
- Added an /admins command, along with an /ahide command for admins to hide themselves from the admin list.
(The full changelog / commit list can be found directly on github)

If you have any questions feel free to ask. Again, I plan on building this script up over-time. The change-log will always be updated when something is changed/a new version is released. Downloads, and other information can also be found at evanabagail.net(no longer the case).

Please do not ask me things like "The MySQL won't connect! Help me" because that has nothing to do with the gamemode; but more of your host / and or lack of MySQL / scripting knowledge / experience. Thank you.

Cheers!
