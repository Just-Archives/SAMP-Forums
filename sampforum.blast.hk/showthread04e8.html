SA-MP Local Server Manager/Script Assistant/Tools

Here is another 'Basic' application for scripters. Has some useful features which will make your things a bit easier.

Video:
[ame]http://www.youtube.com/watch?v=0QYtEvVfXGk[/ame]

Features:
  • ArrayMaker Tool: Allows you to input a text and convert into array easily.
  • Color Code Generator: This tool will create all color definitions/hex etc instantly.
  • PAWN Compiler crash protector: When you crash on compiling, usually you would have to force end the process. But using this tool, you can simply close the compiler and keep pawno editor open without losing any work.
  • Script Managing system: Using variuos tabs, manage your scripts flawlessly. And also find out any requirement problems.
  • Search system: Type a word/function name and search inside the scripts for it's existance.
  • Pawno Crash Handler: When you keep the Manager open in background, it will keep checking for any crashes while compiling. If it takes too long to compile you can shutdown the compiler without damaging your script.
  • And few others..
