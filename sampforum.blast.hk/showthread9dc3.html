San Andreas - Battlegrounds
Beta Version 1.0 - Last Updated: 23 Sept, 2017
Gamemode inspired from popular deathmatch game Player's Unknown battelgrounds ported in SAMP.
Its still in BETA, which means there could be bugs and over all that this gamemode isn't complete at all. I made this a while ago and since i am retired, i thought i could post this here and people might find this interesting and continue the project. Feel free to fork it!

TO-DO
  • Add inventory items all over the map (/newitm, /edititem, /additem)
  • Add vehicles to map
  • Realistic weapon damage
  • Fix lagcomp.inc which is kinda my burden. I'll see when can i do that.
  • Admin commands
  • VIP commands and features if you need so.
  • Extra commands like /anims, /kill, etc.
  • And whatever left.

Future Of This Project
Quote:

This project is NOT DEAD! I have been recently working on updates and basically reworking on the whole mode with some really exciting features and circular forcefield and lot of similar PUBG features. This project was the first PUBG gamemode ever introduced in samp(publicly) and wasn't finished or maintained at all. Now i might have some spare time to work on it again!

I might need some mappers to help me out, any volunteers would be great! Your work will be include little mapping here and there and placing loots in areas.
