First make sure your Server CFG File is setup correctly. It should already be but if not:
Code:
echo Executing Server Config...
lanmode 0
rcon_password <3lol
hostname Your Server Name
gamemode0 CG-RP1
filterscripts buttons Toll ls_elevator AFK gCamera flash lsmall
plugins streamer
chatlogging 0
weburl classified-gaming.tk
onfoot_rate 40
incar_rate 40
weapon_rate 40
stream_distance 500.0
stream_rate 1000
logtimeformat [%H:%M:%S]

 
30 
1 
port 7777maxplayers 30
query 1
announce 
ibind 25.43.187.12
If that doesn't work,

Re-Download the Gamemode. Then go to the SA-MP website and download the latest server. Then copy the contents over the old contents in the main directory except the Server CFG File.

Quote:
Originally Posted by killerjam

How can i make my self owner of all the fractions and how to work or the factions
When you first register exit, go into scriptfiles and find your character name. Then edit in notepad and set your admin level to 100000. Then /makeleader to make yourself a leader of a faction. /editfaction to edit your factions.

As an example, if you wished to change the name of a faction, say #1, then type /editfaction 1 name [name you want here]


hope this helps.
