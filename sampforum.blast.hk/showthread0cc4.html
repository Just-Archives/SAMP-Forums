SA-MP 0.3d R2 client/server update (optional)
Kalcor
SA-MP Developer
Posts: 1,180
Threads: 55
Joined: Apr 2005
Reputation: 0
#1

09.12.2011, 10:15
SA-MP 0.3d R2 update is available on the Download Page. This is an optional/recommended update to SA-MP 0.3d that fixes some last minute problems not found before release.

Client:

- Fixes a common client crash related to the object limit increases.

Server:

- After a player slot was kicked, the server may not properly free the slot. The slot could not be used by other players. This problem has been resolved in the 0.3d R2 server.

SA-MP is also making available an 800 player slot server. It is not recommended that you use the 800 player slot server unless your server requires it, since the script will need to be compiled against MAX_PLAYERS 800, and it may require specially compiled plugins.

0.3d R2 800 slot server downloads:
Windows Server: http://files.sa-mp.com/samp03dsvr_R2-800p_win32.zip
Linux Server: http://files.sa-mp.com/samp03dsvr_R2_800p.tar.gz

Please check back for any further updates to the SA-MP client/server.
