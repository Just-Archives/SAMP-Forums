How to:

Download any php + mysql dev server. Such as EasyPHP, Xampp, wamp, UniformServer.
Install/Run it then start mysql.
Open phpmyadmin and make a new database called 'lsrp'.
Now import the sql file 'lsrp.sql' in the 'lsrp' database. It should take a few seconds.
Change your root password to something (this gamemode's default password is 'db' and the mysql default password is blank).

Unzip 'Sublime Web.rar' inside the 'www' folder of your php dev server.
Edit the mysql password in the file 'class\assets\dbconfig.php'. It should be the same as your mysql root password.

PHP Code:
<?php
/* DATABASE CONFIGURATION */
define('DB_SERVER', 'localhost');
define('DB_USERNAME', 'root');
define('DB_PASSWORD', 'db'); <--- here change the password
define('DB_DATABASE', 'lsrp');
...
?>
Then open the UCP (usually http://localhost/Sublime)
Make a new account.
Confirm your account by email (you wont get an email if your ucp is hosted on localhost) or by changing the 'confirmed' value to 1 in the accounts mysql table.
Set your account as admin by changing the 'admin_level' value to 6 in the accounts mysql table.
Login the UCP.
Now make a new character application.
Accept your character application.

Now unzip the 'Sublime Game.rar' anywhere.
Edit the mysql password by editing the file 'scriptfiles\mysql.ini' . It should be the same as your mysql root password.
PHP Code:
=localhost
=root
=lsrp
=db <--- here change the password 
Start the server.

Now open your samp client and change your nick to FirstName_SecondName (character name, not main account name!) <--- '_' char between.
Join the server and type your main account password.

Now it should be working!
Good luck!
