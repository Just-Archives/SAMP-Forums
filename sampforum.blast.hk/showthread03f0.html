if you dont have OnPlayerInteriorChange in your GM, just put this:

Код:
public OnPlayerInteriorChange(playerid, newinteriorid, oldinteriorid)
{
	Wints_OnPlayerInteriorChange(playerid, newinteriorid, oldinteriorid);
	return 1;
}
