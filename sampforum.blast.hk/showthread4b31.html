Hi all,
Here's my admin script that i created with some infos
General :
-To be admin it's necessary to be logged as rcon admin
-There are 62 commands in the script
-There are a ping kicker, the ping is set to 500, you can change this in the script
-Leave and join messages are in the script
-Admin chat is writen in a .txt file
-Reports are too writen in a .txt file
-Bans & Kicks too
-Admin can read other players PM's

Commands :
Regular player :
/report [Name] [Reason] = report some one
Admin :
/goto [ID] = go to a player
/gethere [ID] = get a player to your position
/akill [ID] = kill a player
/force [ID] = force a player to change class
/rfreeze [ID] [Seconds] = freeze a player for seconds
/freeze [ID] = freeze a player
/unfreeze [ID] = unfreeze a player
/rmute [ID] [Seconds] = mute a player for seconds
/mute [ID] = mute a player
/unmute [ID] = unmute a player
/rjail [ID] [Seconds] = jail a player for seconds
/jail [ID] = jail a player
/unjail [ID] = unjail a player
/explose [ID] = explose a player
* R@f slaps [ID] = slap a player around a bit with a large trout.
/getinfo [ID] = get the player infos in text draw exemple1 exemple2
/sinfo = server infos in text draw
/infooff = switch the server or player info off
/remweapon [ID] = remove a player weapons
/eject [ID] = eject a player
/rspec [ID] = spec a player (from Keyman's adminspec)
/rspecoff [ID] = stop to spec a player
/fuck [ID] = fuck a player [Skin 10, Money 0$, Remove weapons, Wanted level 6, Remove armour, Set health 1, Remove from a vehicle, Set name to Fuckedup] (taked from SA-MP forum, i don't know who created it)
/setname [ID] [New Nick] = set name of a player
/say [Text] = talk as admin [exemple : [RAS]Admin [WSG]Raf : i'am admin ]
/announce [Text] = write a gametext
/kick [ID] [Reason] = kick a player
/ban [ID] [Reason] = ban a player
/sethealth [ID] [Health] = sethealth of a player
/setarmour [ID] [Armour] = setarmour of a player
/skin [ID] [Skin] = set skin of a player
/giveweapon [ID] [Weapon] [Ammo] = give a weapon to a player
/givecash[ID] [Cash] = give cash to a player
/sound [ID] [Sound ID] = set a sound for a player
/setwanted [ID] [Wanted level] = set wanted level for a player
/gravity [Gravity] = set gravity
/serverpass [server pass] = set a server pass
/servername [server name] = change the host name of your server
/servermap [server map] = change the map name of your server
/serverurl [server url] = change the url adress of your server
/healall = heal all players
/amourall = armour all players
/freezeall = freeze all players
/unfreezeall = unfreeze all players
/akillall = kill all players
/kickall = kick all players
/fuckall = fuck all players
/remweaponall = remove all players weapon
/ejectall = eject all player from they vehicles
/flip = flip your car
/rweapon = open a weapon menu
/rmoney = open a money menu
/rtime = open a time menu
/rweather = open a weather menu
/rcar = open a vehicle menu
/rcolor = open a color menu
/rtunning = open a tunning menu
/god = god mode and give you a minugun
/debug-on = debug mode on (rename your debug file to : DEBUG )
/debug-off = debug mode off
/rcmd = see all admin commands
*[Text] = admin chat
