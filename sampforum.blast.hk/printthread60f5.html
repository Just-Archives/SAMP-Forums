Project DayZ SA:MP v15.0 - Kitten - 17.05.2020



The official DayZ SA:MP script created back in 2012. The player base over 100 players on daily basis average hitting nearly almost 200 players. This gamemode was based on the DayZ Arma II mod and not based on the actual DayZ game. I've updated the gamemode from version 14.0 which was last deleted on the forums due to inactivity. Version 15.0 includes some HUD changes/updates, minor bug fixes and new items such as dirty food items. It also includes zombies using FCNPC and custom mapping around San Andreas I hope you all enjoy this! I will update this mode from time to time visit pgagaming.net for updates if not posted here. I will also have a GitHub for this project. For previous version please visit pgagaming.net.

History:

Version 15.0:

- FCNPC Zombies up to 400 zombies
- HUD Upgrades
- Custom Mapping
- New Items
- Minor/Major Bug fixes
- MySQL only. (instructions on GitHub)
- Crate system (/spawncratem (admin CMD) or use DayZSA_CreateAirdrop(x,y,z) function)


Plugin's Required (all on GitHub repo):

- FCNPC
- Sscanf
- Streamer
- MapAndreas
- MySQL (R39-4)
- Whirlpool
