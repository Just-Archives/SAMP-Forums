Player Control Panel
Hey, i came back with a new PCP version. This version is basically the same with PCPv3, only it has a different system. I build it with the Laravel framework, and should have no security issues.

A readme file with the database explained, and also a database file will be found in the archive availabile for download.

Unfortunatelly, Laravel framework only works on PHP 5.3.3, so it will not work on free hostings like 000webhost, because it uses PHP 5.2.*.

Pictures:

http://imgur.com/PmMl0Cp,XqCmKBW,rF6...SoXV,68ylWxf#6

Changelog:

v4.2
*You no longer need public_html access for installation. I moved laravel folder inside main folder, for installation, just upload the files on FTP and the database on phpmyadmin. For other info, please read the "readme.txt" file.

Downloads:
  • PCPv4.2
  • PCPv4.1
  • PCPv4.0
Future plans: i plan to make bug fixes and add minor features, and also to implement the Ace Admin template [****** it]


Extra: if you think your server deserves better and want a premium PCP, PM me or check out my description.
