Hello,

This is an interior map I've just completed. Took around 4-5 hours start to finish. This map is apart of the Welcome To Blueberry project I'm working on which you can see here. This is one of the stages of creating a new busting town with Blueberry. The local motorcycle club's clubhouse interior is matched to the exterior building as best as possible and offers some creativity for roleplay inside and out. I've included an old faction of mine into the map to show a true representation of the clubhouse feel. Map will not be released as it's part of an on-going project.

Note: There are no texture mods/modding to my game screenshots. Everything you see are custom models and textures imported into the game with the appropriate files. Object count 235.
