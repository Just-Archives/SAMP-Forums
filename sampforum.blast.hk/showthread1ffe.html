Quote:
Originally Posted by mooman

The RW section ID for the collision in the DFFs is meant to be 0x0253F2FF, right? On gtamodding it says collision sections use 0x0253F2FA but this conflicts with what kdff generates.
SA-MP has its own collision extension because the game's one has limited buffer space and looks to be insecure. The SA-MP one only supports COL3 format. The game's dff collision loader is disabled when loading dffs for SA-MP, so attaching a col outside kdff wouldn't have any effect.
