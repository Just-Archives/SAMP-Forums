Alright everyone having Problems with 80% loading crash, do this.
1)Right click on SA-MP and Set (Also Tick)Compatibility to Windows 96/ME.
2)Apply -> Ok.
works fine for me.
Note: Doing it will change the location of User Profiles... + Audio Plugins (Songs etc) will have to be Re-Downloaded (If it's available in your Server) Because it also changes Audio Plugins location...
