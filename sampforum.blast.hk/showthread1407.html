2ndly i have left a 2 dialog fuctions empty:

2nd on /help dialog ->>otherhelp - i left it to fill it by your own and add some things that you think are not fitting to other things.
Now How to to change the name of your server through the script:
-go to pawno.
-open the nbtdm.pwn.
-try to find a callback named "ChangeHostname() (version 1.4b - line 24649)
-this is changing the name of the server every 3 seconds.
-change the names on "case0:,case1:,etc... with your name of the server
-do the same on map change and gamemodetext callback i made.
