Hello sa-mp community,
I'm about to create a UCP for SAMP which I would like to publish for free then.

For this reason, I just wanted to ask, what functions etc. you want for a UCP.

The UCP is designed for Reallife / Roleplay servers. Once it is finished there will be another one for DM server etc.

At the moment the UCP is only available in German. Once it is finished, it will also be available in English.

Current functions:
- Webinstaller
- Build with Bootstrap
- UCP color can be changed
- Whirlpool Encryption
- News System
- Server Stats
- Server Information (Status, Players, Playernames, IP)
- Teampeak verification
- Account information (Rank, Banstatus, Premium, Warns, Faction, Money, Level, Kills / Death, Crimes, etc.)
- Namechange System (admins can give a player permission for the name change, then the player can change his name in the UCP)
- Change Password
- Ban administration (ban and unban players + list of banned players)
- Dynamic signature & profile image to download

Scheduled Features:
- Multilingual UCP, languages ​​can easily be added.
- Report players
- 2 start pages to choose from (1x only login & 1x with news, serverinfo & login)
- Faction management
- Page for managing the settings
- Online Banking

What functions would you wish?
Feedback / suggestions are also welcome.

Excuse me if my English is not so good, I hope you understand everything.
