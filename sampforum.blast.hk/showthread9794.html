SAM[P]CE
a code editor for SAMP (well, errm PAWN)
About
San Andreas Multiplayer PAWN Code Editor or SAM[P]CE : advanced multi-PAWN code editor (SAM[P]CE) is yet another code editor, created to ease the process of developing PAWN scripts for SAMP. This editor has many features and they can also be expanded through the use of plugins.

I created this code editor, due to the lack of PAWN editors, especially for SAMP. Most of them however, are no longer developed or lack features that are useful. Also, another reason is due to the fact that many applications released for SAMP, like teleport generators and whatnots, should not be separate applications at all! They could be easily incorporated and given direct access to the code editor itself! That explains the plugins system, by the way.

Features
  • Syntax highlighting
  • AutoComplete & Call tips
  • Function Explorer (includes & opened files)
  • MDI (Multiple Documents Interface)
  • Error listing
  • Expand functionality using plugins

Download
Latest version: 1.0
Download from: CodePlex (select the latest version)
You will need the .NET Framework 3.5, which can be downloaded from the Microsoft website. (No links, as I don't want to commit myself to a download link)

If you are using a 64 bit system, download SciLexer.dll for x64 and replace the SciLexer.dll file with the downloaded version.

The download contains the main app plus two sample, simple plugins, the Colour Picker (C#) and the Statistics Tool (VB.NET).
Mirrors are NOT accepted, due to it being an application and the integrity might be lost.

NOTE: Oh yeah, to activate autocomplete, press CTRL+SPACE.

FAQ

Q My SAM[P]CE crashes on start up!
A If you are using a 64bit OS, please download SciLexer.dll for x64 and replace the SciLexer.dll file in your installation.

Q Some includes do not work properly! Help!
A It is mostly due to the SAM[P]CE parser not parsing the include correctly. It may be due unique coding by the developer. A temporary fix is to remove the file from the include folder, change the function detection to pawno or backward compatible detection (See settings).
