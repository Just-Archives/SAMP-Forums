I just wanted to share this simple e-tool developed by me(the base system showing the objects and base CSS was by Zune_ however I added everything else, aswell as added support for objects beyond the first RC). It basically has images of all the new objects, along with their ID on one simple page!

That's basically it! Enjoy
https://evanabagail.net/objects/

Also accessible from:
http://objects.evanabagail.net

UPDATE: The object viewer now supports RC2 objects.(As of the 28th)
UPDATE2: As of March 25th the viewer now supports RC3 objects.

EDIT:
I am going to leave this information here for additional reference:
You can click "Show All, Show RC1" etc to target a certain RC version and only objects belonging to that version will appear.

You can click the different view angles to see the objects in a different angle.

All objects are rendered through map editor, and any missing objects are usually the result of a fail within map editor(not that I have spotted any missing objects, however the skins don't work so maybe some objects won't either).
