Survival the Island

Powered by Power
Introduction

The gamemode is quite incomplete, contains some legal systems and some bugs as well. I'm giving up on him because I'm out of time, working the day and studying the night. Use it however you want, it's a good base for those who are starting, I'm a beginner in pawn so if gamemode is badly done, forgive me, but that's how I learned!


What's in the gamemode?


Code:
System to create clan [needs 5 cash] with commands / invite / kick off, maximum members per clan are 15.
To pick up wood you hit the tree with the knife, each hit = 1 Wood.
To get stones and metals you hit the rock with the racket, each hit = 1 stone + 1 Metal.
To get bolts, you will have to take metal and use the furnace to turn them into bolts.
The base / home build is now all done by EditObject, making it look more like RUST.
All objects you create can be moved with EditObject.
The Crafting Table used on the server is the bench, in it you create armaments, medicines, among others.
System of killing and beheading the "Bambi" in it you get 2 meats and 3 Cloths.
Meat / fish roasting system in the furnace, in case you eat raw food you lose 5 life.
Saving of all created objects.
When you die, you are set in your bed just like rust.
Inventory system in textdraw.
System registry / login in textdraw.
