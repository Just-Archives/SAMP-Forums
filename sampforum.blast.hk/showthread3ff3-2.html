Welcome to the first ever filterscript that allows players to see the streets that they are on without having to download and mods to gain advantages etc. Players can use this filterscript without having to install any mods on there gta3.img.

This project was started in April and commenced during September 2013. This script will not be released until Version 0.2.0, If you have any enquires or ideas you can post them below

Version 0.1.4
* Added textdraw formatting
* Added in the distance for being close to a certain destination
* Added 5 street locations
* Reformatted the Whole code, for more efficiency
* San Pedro Ave & Hill Street Added
* Extended the variables to 9000 Coordinates (If more is needed will be extended more)

Version 0.1.5
* New Textdraw Position and formating
* Aided support from Ikey07 for the Major bug with Global Textdraw
* 9 Streets Completely Finished and bug free
* Whole new code formatted
* Added to a live testing server.

Filterscript Source will be released upon the full version being completed - I do not like giving out buggy codes

Version 0.1.5 Video

[ame]http://www.youtube.com/watch?v=TtPfXZwGjnY[/ame]
(If you need any help do not ask me, it will be explained below how to make it)

To add a street
1. First go to the crossroad or the end of the street and type /save [Crossroad or street name]
2. take 10 Jogging sprints forward not Sprinting
3. Then type /save [Street name]
4. Go into the code and add the streets.

Please take note you are using the X,Y,Z ratios
